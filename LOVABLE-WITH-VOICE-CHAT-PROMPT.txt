TECHNICAL SPECIFICATIONS
====================

VOICE CHAT IMPLEMENTATION:
Use Web Speech API for voice recognition:
- window.SpeechRecognition or window.webkitSpeechRecognition
- Continuous: false (single utterance)
- Language: en-US
- Interim results: true (show while speaking)

Voice synthesis for AI responses:
- window.speechSynthesis
- Female voice preferred
- Rate: 1.0 (normal speed)
- Pitch: 1.0 (normal)

AI Chat Logic (placeholder - connect to backend later):
- Keyword matching for demo:
  * "trend" ‚Üí Show trending info
  * "color" ‚Üí Show color palette

- Friendly, fashion-focused responses
